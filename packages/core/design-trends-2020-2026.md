> 각 트렌드에는 ① 시각 특징(색/타이포/레이아웃/모션/장식) ② 기존 카탈로그 slug와의 매핑(있으면)을 표기한다.

## C. 카탈로그 갱신 요약 (이 리서치가 시사하는 것)

### 기존 23개 후보로 이미 커버되는 트렌드
- 깊이/material: 글래스모피즘(21·24) → `glassmorphism-aurora-01`, 뉴모피즘(21·26) → `neumorphism-soft-01`.
- 노스탤지어: Y2K(22·26) → `y2k-futurism-01`, 바이퍼웨이브/글리치(20·23) → `vaporwave-synth-01`, 메피스/레트로 → `memphis-postmodern-01`/`retro70s-warm-01`.
- 구조/raw: 네오브루탈리즘(22·25·26) → `neobrutalism-editorial-01`.
- 표현/에너지: 맥시멀리즘·도파민(22·23·26) → `maximalism-dopamine-01`, 콜라주(20·26) → `collage-scrapbook-01`.
- 테크/다크: 오로라 그라디언트(25) → `aurora-gradient-01`, 다크 모드 → foundation `dark`.
- 타이포/편집: 빅 타이포(25)·실험 타이포(23) → `swiss-international-01`/`editorial-magazine-01`.
